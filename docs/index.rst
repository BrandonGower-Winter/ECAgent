Welcome to ECAgent's documentation!
===================================

`ECAgent`_ (Entity-Component Agent) is a Python 3+ Agent-Based Modelling (ABM) framework.

ECAgent offers a suite of tools (such as spatial environments
and data collectors) that enables users to create a wide variety of ABM.

The easiest way to get started with ECAgent is to check out the `Introductory Tutorial`_.
